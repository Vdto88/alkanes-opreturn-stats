# Figures

`gen-fig-13.py` renders every chart from the "Alkanes by the Numbers" article
(and the fig-site set mirrors subfrost.io/metrics) straight from `../history.csv`.

```bash
python gen-fig-13.py   # Python 3, stdlib only. Prints the headline numbers and writes the SVGs.
```

This folder is published to the public data repo (Vdto88/alkanes-opreturn-stats)
on every daily run, next to the `history.csv` it reads.
